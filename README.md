# SkillForge Academy - Landing Page

## 📌 Project Overview

This project is a responsive **Landing Page** developed using **HTML5** and **CSS3** as part of the **CodSoft Web Development Internship – Level 1 Task 2**.

The landing page is designed for **SkillForge Academy**, an online learning platform that offers programming and web development courses. The website features a clean, modern, and user-friendly interface with responsive design.

---

## ✨ Features

- Responsive Design
- Attractive Navigation Bar
- Hero Section with Call-to-Action Button
- Features Section
- About Section
- Call-to-Action (CTA) Section
- Modern UI with Gradient Colors
- Footer

---

## 🛠️ Technologies Used

- HTML5
- CSS3
